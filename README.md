# Evomon Auto-Hunter

บอทเดินหามอนเองในเกม Roblox Evomon → สู้ → จับ → **เจอ Shiny/Prismatic = หยุดทันที + เสียงเตือน** (ไม่กด E ให้คุณจัดการเอง)

---

## เปิดใช้งาน
**ดับเบิลคลิก `start.bat`** — ขอสิทธิ์ Administrator ให้เอง (สำคัญ! ไม่งั้นปุ่มไม่เข้าเกม) + **ลง dependency ให้อัตโนมัติรอบแรก** (รวม OCR) แล้วเปิด GUI

หรือเปิด **PowerShell แบบ Run as Administrator** แล้วรันเอง:
```
py -3.12 -m pip install -r requirements.txt   # ครั้งแรกครั้งเดียว
py -3.12 gui.py
```
> รันแบบไม่มีหน้าต่าง: `py -3.12 main.py`
> OCR ใช้ RapidOCR (CPU) — โหลดโมเดลครั้งแรกช้านิดนึง บอทอุ่นเครื่องให้ตอนกดเริ่ม

ในหน้าต่าง: ตั้งค่าสกิล → กด **▶ เริ่ม** | **F8** = หยุดฉุกเฉิน (ใช้ได้แม้อยู่ในเกม)

---

## flow การทำงาน
```
SEARCH  หามอนในจอ (ป้าย "Lv." ตัวอักษรขาว)
  ├─ ไม่เจอ → กวาดกล้อง + เดินหา + ตั้งมุมกล้องใหม่เป็นพักๆ
  └─ เจอ   → SEEK
SEEK    เดินเข้าหามอน (กดปุ่มทิศค้างต่อเนื่อง ลื่นไม่กระตุก)
BATTLE  เจอป้าย "TIME" → วนกดสกิล (เลข) ซ้ำจนกว่าจะถึงหน้า Catch
CATCH   หน้า Catch:
  ├─ ปกติ (เจอทั้ง Prismatic+Shiny) → กด E จับ (วนจนได้/พลาด → หาตัวใหม่)
  └─ Shiny/Prismatic (แถวในแผงหาย) → หยุดทันที + ไม่กด E + เด้งแจ้งเตือน
```

---

## วิธีตรวจ Shiny/Prismatic (หัวใจ)
> สีตัวมอนเปลี่ยนตาม **ธาตุ** (ไฟ=แดง หญ้า=เขียว) ใช้สีตัดสิน shiny ไม่ได้

ใช้ **OCR (RapidOCR)** อ่านแผง Obtain Rate (มุมซ้ายล่าง) บนหน้า Catch — ดู **"ค่าอัตรา"** ของแต่ละแถว:
- ปกติค่าเป็น **ตัวเลข** เช่น `Shiny: 0.2%`, `Prismatic: 0.45%`
- **แถว Shiny ค่าเป็น `--`** → ตัวนี้ = **SHINY**
- **แถว Prismatic ค่าเป็น `--`** → ตัวนี้ = **PRISMATIC**

วิธีให้เร็ว (det model ของ OCR ช้ามากบนบางเครื่อง):
- ตรวจ "อยู่หน้า Catch ไหม" ด้วย template กล่อง `obtain_panel.png` ก่อน (เร็ว ระดับ ms)
- พอถึงหน้า Catch ค่อยครอปบรรทัด Shiny/Prismatic ส่งให้ OCR **อ่านอย่างเดียว (rec-only)** ~200ms
- ทดสอบกับ shiny.png/catchscreen.png แล้ว แยก shiny/normal ถูก 100%

เจอแล้ว → หยุด + บันทึกประวัติอัตโนมัติ + เสียงเตือน (**ไม่กด E** ให้คุณจัดการเอง)

## ฟีเจอร์
| ฟีเจอร์ | รายละเอียด |
|---------|-----------|
| เดินหามอนเอง | ตรวจป้าย Lv. แล้ว**ค่อยๆ เดินเข้าหา** แตะ w/a/s/d ทีละก้าวแบบคน (ไม่พุ่ง/ไม่กดค้าง) |
| สู้อัตโนมัติ | กดสกิลเลข 1-3 (+ สกิล 4 อัตโนมัติ) ซ้ำจนถึงหน้า Catch |
| **ไม่จับเอง** | ไม่กด E — สู้เสร็จก็เดินหาตัวต่อไป (หยุดเฉพาะตอนเจอ shiny/prismatic) |
| ตรวจ Shiny/Prismatic | OCR อ่านแผง Obtain Rate (ค่า `--`) แม่นเกือบ 100% ไม่ใช้สี |
| Full Window | บังคับ Roblox เป็นหน้าต่าง maximize (ไม่ใช่ fullscreen exclusive) |
| กล้องล็อกมุม | ตั้งมุมปกติ (ตัวอยู่กลางจอ) อัตโนมัติ + ล็อกกลับองศาเดิมถ้าถูกหมุน |
| เฉพาะ Roblox | ทุกการกดเช็คว่า Roblox โฟกัสก่อนเสมอ + ดึงโฟกัสกลับถ้าหลุด |
| ประวัติ | เก็บ Shiny/Prismatic ที่เจอใน `history.json` |

## ไฟล์ในโปรเจกต์
| ไฟล์ | หน้าที่ |
|------|---------|
| `gui.py` | **หน้าต่าง UI (ตัวที่ใช้รัน)** |
| `bot.py` | สมองบอท (เดิน/สู้/กล้อง/โฟกัส — แยกเธรดตรวจภาพ+ควบคุม) |

| `roblox.py` | ตรวจ/โฟกัส/maximize หน้าต่าง Roblox (Windows API) |
| `config.py` | ค่าตั้งทั้งหมด (ค่าส่วนตัวเขียนทับใน `config_user.py` — ไม่โดนลบตอนอัพเดต) |

| `update.py` | อัพเดตเวอร์ชันใหม่จาก GitHub อัตโนมัติ (start.bat เรียกให้เอง) |

### Template (รูปอ้างอิงที่ใช้ตรวจจับ — เหลือเท่าที่ใช้จริง)
| ไฟล์ | ใช้ตรวจ |
|------|---------|
| `monster_lv.png` | ป้าย "Lv." ของมอน (ครอปจาก mob1 จริง) |
| `templates/lv/*.png` | ป้าย Lv. แบบเพิ่มเติมที่ "เรียนรู้" จากภาพมอนจริง (สร้างโดย `learn_mobs.py` บอทโหลดเองอัตโนมัติ) |
| `battle_time.png` | ป้าย "TIME" (= เข้า battle) |
| `obtain_panel.png` | กล่อง "Obtain Rate" (= อยู่หน้า Catch, gate ก่อนยิง OCR) |

> shiny/prismatic ไม่ใช้ template แล้ว — ใช้ **OCR อ่านค่า** (`ocr.py`) แทน จึงได้ prismatic อัตโนมัติ

- `config_user.py` — ค่าตั้งส่วนตัวของเครื่องนี้ (สร้างเองถ้าอยากแก้ค่า — ไม่โดนทับตอนอัพเดต)

## สอนมอนเพิ่ม (บอทหามอนบางตัวไม่เจอ)
1. เซฟภาพหน้าจอที่เห็นตัวมอน + ป้าย Lv. ชัดๆ เป็น `mob1.png`, `mob2.png`, ... วางในโฟลเดอร์บอท
2. รัน:
   ```
   py -3.12 learn_mobs.py
   ```
3. สคริปต์จะเช็คเองว่าจำมอนตัวนั้นได้หรือยัง:
   - จำได้ดีแล้ว → ข้าม (ไม่เพิ่ม template ซ้ำให้บอทช้า)
   - จำได้ไม่แม่น → ครอปป้าย Lv. จากภาพเก็บเข้า `templates/lv/` ให้เอง
   - หาป้ายไม่เจอเลย → บอกให้ครอปเองด้วย `make_template.py`
4. อยากแจกให้เพื่อน → commit + push แล้วขยับเลข `VERSION` (ดูหัวข้อถัดไป)

> ทดสอบกับ mob1–7 แล้ว: จำได้หมด (คะแนน 0.84–0.94) และเรียนป้ายขนาดใหญ่จาก mob3 เพิ่ม 1 แบบ

## ตั้งค่าสำคัญ (config.py)

### สกิล (ไม่จับเอง)
```python
PRIMARY_SKILL = "3"        # สกิลหลักที่วนกด (1-3)
USE_SKILL4 = True          # กดสกิล 4 อัตโนมัติ
SKILL_INTERVAL = 0.9       # เว้นกี่วินาที/กดสกิล 1 ครั้ง
STOP_ON_PRISMATIC = True   # เจอ prismatic ก็หยุดด้วย (ต้องมี prismatic_dash.png)
```

### เดินเข้าหา (ค่อยๆ เดินแบบคน)
```python
STEP_HOLD = 0.14           # แตะปุ่ม 1 ก้าวค้างกี่วินาที (สั้น=ก้าวสั้น)
STEP_GAP  = 0.10           # เว้นระหว่างก้าว (ปล่อยปุ่ม) ให้เหมือนคนเดิน
SEEK_NEAR_DIST = 150       # ใกล้มอนแค่นี้ -> ก้าวสั้นลง (STEP_HOLD_NEAR) กันพุ่ง
```

### ตรวจจับ
```python
MONSTER_THRESHOLD = 0.62   # ความมั่นใจเจอป้าย Lv.
BATTLE_THRESHOLD = 0.65    # ความมั่นใจเจอ TIME
PANEL_THRESHOLD = 0.70     # ความมั่นใจว่าอยู่หน้า Catch (กล่อง Obtain Rate)
OBTAIN_CONFIRM = 2         # ต้องเจอกี่เฟรมติดถึงฟันธง shiny
# ครอปบรรทัดให้ OCR อ่าน (เศษส่วนจอ x0,x1,y0,y1) — จูนถ้าจอ/สเกลต่าง
OCR_SHINY_FRAC = (0.05, 0.20, 0.910, 0.950)
OCR_PRIS_FRAC  = (0.05, 0.20, 0.835, 0.875)
```

### กล้อง (ล็อกมุมคงที่ + ตัวอยู่กลางจอ)
```python
CAMERA_TOPDOWN = False        # False = มุมปกติ ตัวอยู่กลางจอ | True = ก้มจากบน
CAMERA_CENTER_UP_STEPS = 9    # มุมปกติ เงยขึ้นเยอะให้ตัวอยู่กลางจอ (มาก=เงยมาก)
CAMERA_UP_STEPS = 4           # ใช้ตอน TOPDOWN=True เท่านั้น
CAMERA_FIX_EVERY = 4.0        # ล็อกมุมกล้องกลับองศาเดิมทุกกี่วินาที (กันผู้เล่นหมุน)
CAMERA_ZOOM_PRESSES = 9       # ซูมออกกี่ครั้ง (ปุ่ม o ของ Roblox)
CAMERA_DOWN_STEPS = 12        # ดันกล้องลงสุดก่อน (clamp) -> มุมคงที่เป๊ะ
CAMERA_TILT_SIGN = 1          # ถ้ากล้องกลับทิศ เปลี่ยนเป็น -1
```
> หาไม่เจอ Lv. = หมุนกล้องกวาดรอบหา (เดินหน้าเป็นพักๆ ทุก `SCAN_WALK_EVERY` ครั้ง) จนกว่าจะเจอ

### หน้าต่าง Roblox
```python
REQUIRE_ROBLOX = True      # ทำงานเฉพาะตอน Roblox เปิด
FORCE_FOCUS = True         # หลุดโฟกัส -> ดึงกลับเอง
MAXIMIZE_ON_START = True   # Full Window ตอนเริ่ม
```

## ปรับจูน (อาการ → แก้)
| อาการ | แก้ |
|-------|-----|
| หามอนไม่เจอ | สอนเพิ่มด้วย `learn_mobs.py` (ดูหัวข้อ "สอนมอนเพิ่ม") หรือลด `MONSTER_THRESHOLD` |
| ไม่เข้า battle / กดสกิลไม่ติด | ลด `BATTLE_THRESHOLD` |
| ตรวจ shiny ไม่ติด / ติดมั่ว | เลื่อน `OCR_SHINY_FRAC`/`OCR_PRIS_FRAC` ให้ครอบบรรทัดพอดี, ปรับ `PANEL_THRESHOLD` |
| ตัวไม่อยู่กลางจอ (ก้ม/เงยไป) | เพิ่ม/ลด `CAMERA_UP_STEPS` |
| กล้องเงยแทนก้ม | `CAMERA_TILT_SIGN = -1` |
| มุมกล้องเพี้ยนบ่อย | ลด `CAMERA_FIX_EVERY` (ล็อกถี่ขึ้น) |
| มาโครหลุดไปจออื่น | รันแบบ **Administrator** (จำเป็น) |
| สกิลกดเร็ว/ช้าไป | ปรับ `SKILL_INTERVAL` |
| เดินเข้าหาเร็ว/พุ่งไป | ลด `STEP_HOLD` หรือ เพิ่ม `STEP_GAP` |
| เดินช้าไป / ไม่ค่อยขยับ | เพิ่ม `STEP_HOLD` หรือ ลด `STEP_GAP` |

## ข้อควรรู้
- ต้องรันแบบ **Administrator** ปุ่ม/เมาส์ถึงจะส่งเข้า Roblox ได้
- ตั้ง Roblox เป็น **โหมดหน้าต่าง (Windowed)** บอทจะ maximize ให้เอง (ไม่ใช่ fullscreen exclusive)
- สกิลกดเป็น **เลข 1-4** (ใช้ได้ทุกตัวละคร ไม่ผูกชื่อสกิล)
- การใช้มาโครอาจผิด ToS ของเกม เสี่ยงโดนแบน — **ใช้ภายใต้ความรับผิดชอบของผู้ใช้เอง**

## สิ่งที่ยังต้องเทสจริง
- **ตรวจ shiny/prismatic (OCR)**: ทดสอบกับ `shiny.png` / `catchscreen.png` แล้ว แยกถูก ✓ — prismatic ใช้ OCR เหมือนกันจึงเปิดอัตโนมัติ ควรลองในเกมจริงยืนยัน
- **ตำแหน่งครอป OCR**: `OCR_SHINY_FRAC`/`OCR_PRIS_FRAC` จูนจากจอ 1920×1080 ถ้าจอ/สเกล UI ต่าง อาจต้องเลื่อนนิดหน่อย
- **กล้อง/การเดิน**: ลองกด ▶ เริ่ม ดูว่ามุมตัวอยู่กลางจอ + เดินเข้าหาแบบค่อยๆ ก้าว (ไม่พุ่ง) โอเคไหม
